JavaScript DOM - Changing CSS

How cool is this? The HTML DOM allows us to change the style, or CSS, of HTML elements!

HTML DOM - Using Events

Events are generated by the browser when things "happen" to HTML elements, such as:

  • User clicks button
  • Page Loads
  • Input Fields are changed

Let's change a CSS property when a button is clicked. We don't even need a JS file OR script tag for this.
